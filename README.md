# recipe-sharing-hub

A community-driven recipe gallery where users browse, post, and save recipes.

## Features
- Browse all recipes
- Submit a new recipe
- Favorite recipes

## Quick Start
1. `npm install`  
2. Create `.env.local` with your Supabase URL & anon key  
3. `npm run dev`
